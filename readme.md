🏢 Objetivo

Automatizar o processo de geração e impressão de etiquetas de produtos, garantindo:

Padronização das etiquetas
Redução de erros de precificação
Agilidade na atualização de preços
Integração direta com o ERP da empresa
🚀 Funcionalidades
🔎 Consulta de produtos diretamente no ERP
💲 Recuperação automática de preços atualizados
🏷️ Geração de etiquetas em ZPL
🖨️ Impressão em impressoras Zebra
📦 Suporte a múltiplos produtos
⚡ Interface dinâmica com jQuery
🎨 Layout personalizável das etiquetas
🛠️ Tecnologias Utilizadas
PHP (Backend)
Angular.js
JavaScript
jQuery
HTML5
CSS3
Integração com CISS ERP
Linguagem ZPL
⚙️ Como Funciona
O sistema consulta os dados dos produtos no ERP
O usuário seleciona os itens desejados
A aplicação gera o código ZPL automaticamente
O conteúdo é enviado para a impressora Zebra
As etiquetas são impressas prontas para uso nas gôndolas
💼 Aplicações

Este sistema é ideal para:

Supermercados
Lojas de varejo
Farmácias
Estoques e centros de distribuição
